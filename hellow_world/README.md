# Hello World (Flutter)

Minimal Flutter app that displays "Hello world" centered on screen.

## Prerequisites
- Flutter SDK installed: https://flutter.dev/docs/get-started/install
- An emulator or device connected, or use `flutter run -d windows` (on Windows)

## Run
```bash
flutter pub get
flutter run
```

Run on specific device (example: Windows desktop):
```bash
flutter run -d windows
```

Notes
- The main app source is in `lib/main.dart`.
- Use hot reload (`r`) while running to see UI changes instantly.
